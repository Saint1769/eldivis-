
Eldivis - New Year chat (full demo)

Features included in ZIP:
- User registration/login (username + password)
- Editable profile (nickname, avatar, coins shown) - username immutable
- Public user list, open profile with gifts, avatar, coins
- Private chats between users with attachments (images, files, video)
- Daily wheel of fortune for diamonds (coins)
- Gift NFTs to other users
- Block users, delete messages, delete account (from profile)
- New Year themed UI "Eldivis"

Deploy:
1. Create GitHub repo, push this project.
2. On Render create Web Service, connect repo.
3. Build Command: npm install
4. Start Command: npm start
5. Recommended: set JWT_SECRET env variable on Render.
